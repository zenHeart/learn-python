# 数值运算

数值计算中，常见运算符如下:

  • +
  • -
  • *
  • /
  • // 向下取整除法
  • ** 幂运算
  • () 表示分组

在 REPL 模式下，执行如下语句

print( 2 + 2 ) # 4
print( 2 - 2 ) # 0
print( 2 * 3 ) # 6
print( 2 / 3 ) # 0.6666...
print(2 // 3) # 0，0.66... 向下取整返回 0
print(2 ** 3) # 8，幂运算类似 2^3
print( 5 - 1 * 4) # 1， 因为乘法优先级高于减法
print( ( 5 - 1 ) * 4 ) # 16，采用 () 分组，先计算 ( 5 - 1),在乘以 4

# 数值类型

除了整数，python 还支持浮点，复数和其他进制表示法。

  • 浮点用来表示包含小数的数值，采用 . 来分隔整数和小数部分，例如 11.1,10.2
  • 可采用 _ 来分隔数值，例如 1_000_000 的形式来表示大整数
  • 也可采用 科学计数法 表示大整数，例如 1.2e10 表示 12_000_000_000
  • 采用 j,J 来表示复数的虚部，例如 3 + 1j，3 为实部，1j 为虚部 注意虚部前缀必须包含数值，直接写 3 + j 非法。运算符对复数计算也有效
  • 通过添加 0<进制标识> 的形式，来表示其他进制，例如
    • 012 表示 10 进制，python 默认采用 10 进制，所以前导 0 可忽略
    • 0b,0B 表示二进制，例如 0b10 表示 10 进制 2
    • 0o,0O 表示八进制，例如 0o10 表示 10 进制 8
    • 0x,0x 表示十六进制，例如 0x10 表示 10 进制 16

在 REPL 模式下，执行如下语句，熟悉进制表示

# 浮点数
print(1.1 + 1.2) # 1.3
print(-0.23) # -0.23


# 科学计数法 
print(1e5) # 100000
print(1.1e-2) # 0.011
print(1.22e3) # 1220.0

# 复数运算
print(1j + 1j) # 2j
print(1j - 1j) # 0j
print(1j * 1j) # -1 + 0j
print(1j / 1j) # 1 + 0j


# 0b,0B 表示 二进制表示
print(0b10, 0B1011) # 2 11
# 0o,0O 表示八进制表示
print(0o1, 0O1) # 1 1
# 0x,0X 表示 十六进制
print(0x1, 0x1) # 1 1

# 采用 _ 分隔数据
print(1_000_000) # 1000000
# 支持其他进制
print(0b0000_1000) # 8
print(0o000_100) # 64
print(0x00_ff) # 255

# 浮点精度

由于计算机采用二进制表示实际数值，浮点数据的表示会存在差距，所以 0.1 + 0.1 + 0.1 == 0.3 会返回 false, 注意此类问题。

print(0.1 + 0.1 + 0.1 == 0.3) # False

# Strings

采用单引号 '<内容>' 或双引号 "<内容>" 定义字符串。

print('hello world!') # hello world!
print("hello world!") # hello world!
# 嵌套使用单双引号，注意反斜杠表示转义字符
print('"Isn\'t," they said.') # "Isn\'t," they said.

可采用 """<内容>""" 的方式定义多行文本

# 定义多行文本
print("""\
Usage: thingy [OPTIONS]
     -h                        Display this usage message
     -H hostname               Hostname to connect to
""")

# 转义字符

转义字符用于表示不可直接显示的内容例如换行，缩进等。 常采用 \转义符 的方式表示。常见的转义符如下

符号含义
\n回车
\ttab 缩进
\\表示 \
# \n 换行符, 返回
print('test\nbreak line') 

# \t tab 缩进
print("hello\tworld!") # hello   world!

# \\ 表示 \\
print('\\') # \

为了避免 \ 转义可采用 r<字符串> 的模式阻止转义。

# r 阻止转义
print(r'test\nbreak line') # test\nbreak line
print(r"hello\tworld!") # hello\tworld!
print(r'\\') # \\

# 字符串操作

  • 可采用 <字符串>*n 的方式重复某个字符串内容 n 次，n 为正整数
  • 可采用 <字符串> + <字符串1> + ... 的方式连接多个字符串
    • 也可直接使用 <字符串> <字符串1> ... 方式合并字符串
# 字符串*n 重复字符串内容 n 次
print('ba' + 'na'*2) # banana
# 字符串 + 字符串1 + ... 合并字符串
print('ba' + 'na' + 'na') # banana
# 也可省略 + 号
print('ba' 'na' "na") # banana

# 字符串切片

可以采用 [n] 的方式索引单个字符 n 也可以为负值，和字符串对应关系如下

 +---+---+---+---+---+---+
 | P | y | t | h | o | n |
 +---+---+---+---+---+---+
 0   1   2   3   4   5   6
-6  -5  -4  -3  -2  -1

# 字符串[索引] 返回单个字符
print('python'[0]) # p

# 也可采用变量索引
a = 'python'
print(a[0]) # p

# -1 表示最后一个字符
a = 'python'
print(a[-1]) # n

也可采用形如 [start:end] 的方式提取一个字符串片段。start,end 均为可选且支持负值，表示提取从 start 开始长度为 end - start 的字符，注意当 end 为负数时，可将 start 根据索引关系转换为负数处理。

# extract string 
a = 'python'
print(a[0:2])  # py
print(a[-2:-1])  # o

print(a[0:])  # python
print(a[:-1])  # pytho

# 不报错，返回空
print(a[7:9])
# 范围包含整个字符，取所有字符
print(a[-10:9]) # python

可以用 len 方法获取字符串长度

print(len('banana')) # 6
